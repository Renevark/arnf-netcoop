ARNF NetCoop — Online Co-op for A Robot Named Fight
Version 1.0.0
By Etto

This mod adds 2-player online co-op to A Robot Named Fight. It contains NO game
assets: it patches your own installed copy of the game at runtime via BepInEx /
Harmony and does not modify or redistribute any game files.
